    1. 安装并使用主题

    Hexo 默认的主题是名为landscape的一款主题，鉴于每个人喜好不同，也有喜欢这种默认主题风格的，在这推荐一款经过优化以适用于国内网络环境的landscape主题，landscape Plus。

    安装方法大同小异，这里以这款landscape plus 主题为例。

    等待下载安装结束后在整个博客的_config.yml中将theme一行更改为theme: landscape-plus（注意使用英文标点以及冒号后有一个空格）然后生成并部署静态网页hexo d -g即可。

    针对其他不同主题的应用，打开blog/themes/中可以找到对应的主题文件夹，后在博客的_config.yml中theme一行将主题名更改为对应主题文件夹的名字，其余步骤同上。

    除了上述推荐的landscape plus，Hexo的官网上还有着各种各样的主题可供选择。这里简单推荐几个：
    1. Next GitHub上加星第一的一款主题，主打简洁以及广泛的个性化设定
    • 运行代码git clone https://github.com/iissnan/hexo-theme-next themes/next进行安装即可
    1. Yilia GitHub上排名第二的一款简洁优雅的Hexo主题
    • 运行代码git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia进行安装即可

      以及基于Yilia主题修改的双栏Hexo主题Yelee可供选择。
    • 运行代码git clone https://github.com/MOxFIVE/hexo-theme-yelee.git themes/yelee安装即可。
    1. Material 一款扁平风的Hexo极简主题，根据Material Design的风格设计
    • 运行代码git clone https://github.com/viosey/hexo-theme-material.git material安装即可。
    1. Tranquilpeak 一款十分优雅华丽的Hexo主题，自带较多功能，上手容易。
    • 下载最新的代码，后解压到blog/themes/tranquilpeak即可。

    1. Miccal 偶然发现的一个专注于相册等视觉艺术的Hexo主题，效果很棒。

    2. Melody 现在正在使用的一款主题，动画流畅华丽，易于上手。

    • 运行代码git clone -b master https://github.com/Molunerfinn/hexo-theme-melody themes/melody安装即可。

    3. 加入音乐播放器

    在个人网页中加入音乐播放器有以下两种方式：

    • 使用网易云音乐进行添加
    • 使用插件添加播放器

    利用网易云音乐添加音乐播放器

    1. 使用网易云音乐进行添加，首先在网易云音乐官网中找到想要使用的音乐，然后点击生成外链播放器：
    2. 之后选择iframe插件，选择合适的尺寸，是否自动播放，之后复制下方的代码，然后在文章适当的位置将代码复制，运行代码hexo d -g即可部署到网页上。

    利用插件添加音乐播放器

    1. 推荐aplayer音乐插件,这是一款简洁的网页内的播放器插件，可以自定义播放内容，曲目名称、歌词、播放器尺寸以及封面等。运行代码npm install --save hexo-tag-aplayer安装即可。
    2. 在文章的合适页面中添加代码 。可以将曲目和封面与网页一样托管在GitHub或是Coding上，也可以上传到七牛等CDN上进行保存，通过这种方式即可绕过网易云音乐中无版权音乐不能生成外链播放器的尴尬。

    • 此网页是在freenom申领的，个别后缀的顶级域名提供了第一年的免费服务，性价比异常的高。
    • 注册选择好喜爱的域名后，在blog/source/根目录下创建一个文件CNAME，没有后缀的一个文件，打开并添加一行自己注册的域名（不带https://的域名）并保存退出，后在GitHub托管静态网页的repository设置中添加自己的custom domain。
    • 在DNSPOD注册账号，在域名解析中添加域名。
    • 域名添加完成后按照如下内容添加，其中192.30.252.153和192.30.252.154为GitHub提供的托管网站的IP地址，直接填入即可，记录下类型为NS的两条记录值；主机记录为www的一行记录值是在GitHub上托管静态网页的地址。
